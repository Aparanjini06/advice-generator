# Advice Generator App 

This is a simple Advice Generator App that shows random advice when you click the dice button.

---

Features

- Shows advice text
- Changes advice number
- Dice button to generate new advice
- Simple and clean UI
- Uses Font Awesome icons

---
Technologies Used

- HTML  
- CSS  
- JavaScript  
- Font Awesome  

---
Project Structure
  advice-app/
  │
  ├── index.html
  ├── style.css
  ├── script.js
  └── README.md
  
How to Run

1. Download or clone this repository  
2. Open `index.html` in your browser  
3. Click the dice button to get advice
